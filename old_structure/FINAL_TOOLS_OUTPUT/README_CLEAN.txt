================================================================================
DATA FORMAT
================================================================================

Columns (8 total):
  Date     - Calendar date (YYYY-MM-DD format)
  Time     - Time of release
  Currency - Currency affected (USD, EUR, AUD, etc.)
  Impact   - Impact level (high, medium, low, unknown)
  Event    - Event name
  Actual   - Actual released value
  Forecast - Forecasted value
  Previous - Previous period value

Example Row:
  2025-01-06,2:30pm,EUR,medium,German Trade Balance,16.7B,15.3B,16.9B
  2025-01-08,8:30pm,USD,high,FOMC Member Speaks,,,

================================================================================
IMPACT CLASSIFICATION
================================================================================

üî¥ HIGH
  - Central bank decisions (FOMC, ECB, BOE, etc.)
  - Major employment data
  - GDP releases
  - Inflation reports
  - Examples: "FOMC Member Speaks", "Nonfarm Payroll", "CPI"

üü† MEDIUM
  - PMI reports
  - Factory orders
  - Consumer confidence
  - Durable goods
  - Examples: "Manufacturing PMI", "Factory Orders"

üü° LOW
  - Speeches (non-decision)
  - Holiday closures
  - Surveys
  - Regional data
  - Examples: "Speech", "Holiday", "Sentiment"

‚ö™ UNKNOWN
  - Could not classify
  - New event types
